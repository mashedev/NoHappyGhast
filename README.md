# NoHappyGhast
NoHappyGhast is a lightweight and performance-friendly plugin that allows you to completely disable Happy Ghast (flying balloon) entities spawning on your server.  With this plugin, you can easily prevent unwanted Happy Ghast spawns and keep your gameplay experience under control.

Features;

• Blocks Happy Ghast entities from spawning on the server
• Prevents placing or using Happy Ghast related items
• Blocks interactions with Happy Ghast entities
• Optional automatic removal of already existing Happy Ghasts
• Explosion, piston and environment damage protection
• Player notification system when an action is blocked
• Permission bypass support (nohappyghast.bypass)
• Keyword based detection for cross-version compatibility
• Per-world support (enable or disable the system for specific worlds)
